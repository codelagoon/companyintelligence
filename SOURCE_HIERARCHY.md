# Source Hierarchy

Version: 1.0.0

## Purpose
Define the canonical ranking of evidence sources used by Hermes when researching companies.

## Tier 1 — Primary Sources
- Regulatory filings (SEC, FCA, etc.)
- Official company website
- Investor presentations
- Earnings calls
- Official press releases
- Trust centers
- Official documentation

## Tier 2 — Direct Operational Sources
- Product documentation
- API documentation
- Engineering blogs
- Status pages
- GitHub repositories
- Changelogs

## Tier 3 — Independent Professional Sources
- Reputable industry publications
- Analyst research
- Customer case studies
- Partner announcements

## Tier 4 — Structured Intelligence
- Hiring data
- Patent databases
- Company databases
- Technology fingerprinting

## Tier 5 — Weak Signals
- LinkedIn
- Reddit
- X
- Glassdoor
- Community forums

## Rules
- Prefer higher-tier evidence whenever available.
- Use lower-tier sources only to generate or validate hypotheses.
- Never elevate weak signals above primary evidence without corroboration.
